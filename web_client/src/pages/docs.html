API Documentation

Gunakan x-api-key untuk akses endpoint currency.

Currency API

GET /api/v1/currency/latest

Query: base, symbols

GET /api/v1/currency/latest?base=USD&symbols=IDR,EUR
Header: x-api-key: sk_live_xxx

GET /api/v1/currency/convert

Query: from, to, amount

GET /api/v1/currency/convert?from=USD&to=IDR&amount=10
Header: x-api-key: sk_live_xxx
